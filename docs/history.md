Xeno Bot's history

----

::timeline::

- title: 26th Feb 2026 - Xeno Bot creation
  content: |
    Repository created and initial bot scaffolding committed. This includes the
    command framework, spawn system and basic event handlers that form the core
    of Xeno Bot's functionality.
  icon: ':material-clock:'

- title: 27th Feb 2026 - Initial commands & docs skeleton
  content: |
    Added the first set of commands and documentation skeletons for admin,
    gameplay and collection features. Early contributors began iterating on
    command behaviour and configuration.
  icon: ':material-rocket:'

- title: 04th Mar 2026 - Documentation site & tooling
  content: |
    Added an MkDocs site using Material for MkDocs. Implemented build-time
    fixes and automation: corrected CSV paths, fixed a template fallback in
    the footer, and added pre-build generators for contributors and backlinks
    so documentation pages can include up-to-date author and pathway data.
    Also added the Predalien xenomorph page and related host documentation.
  icon: ':material/book-open:'

- title: 09th Mar 2026 - Presentation and accessibility improvements
  content: |
    Improved documentation presentation: added inline image utilities and
    responsive CSS tweaks, adjusted navigation icons to valid Material names,
    and cleaned up generated fragment includes. Continued work on automation
    and build reliability.
  icon: ':material/tune:'

::/timeline::
